=pod

README for Getopt::Gen

=head1 ABSTRACT

Perl module for extended gengetopt-like parsing and
C source generation.

=head1 REQUIREMENTS

=over 4

=item * Parse::Yapp

=item * Parse::Lex

=item * Text::Template

=back

=head1 DESCRIPTION

Getopt::Gen is a perl module which generates C code for command-line
utilities.  It accepts a superset of the command-line parser specifications
used by gengetopt(1).
